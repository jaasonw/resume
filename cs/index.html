Return Values vs Printing

A common mistake I see people make when learning functions is distinguishing the difference between a return value and a function that prints something 

int sum(int a, int b) {
  return a + b;
}

// not the same!
void sum2(int a, int b) {
  cout << a + b;
}

Return values

Return values are what a function call would spit out after it successfully runs.
